How to create pages

The headings are used for splitting up the content file. An H1, H2 or H3 heading will dynamically split the document into new pages in the table of contents. An H4-heading is used as a heading within a page. See the following pages...

An H2 heading splits into a new page

An H3 heading

An H3 heading splits into a new page

An H4 heading

An H4 heading does not by default split into a new page

How to delete a page

When you delete a document's heading or an entire page, normally an empty heading is left in the document - remove this by using the paragraph button or by using the HTML/Layout button to get hold of the HTML-code for the heading, which is to be deleted.
